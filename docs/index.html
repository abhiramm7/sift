Sift is a fast, native macOS app for engineers and researchers to collect → tag → rate → recall papers, books, and reports. Everything lives as plain files in a folder you control. No accounts, no server, no subscription.

Reference managers got heavy. Sift didn't.

Most working researchers don't need a citation engine day to day. They need to collect a paper the moment they find it, tag it so it's findable, rate it so the good stuff floats up, and recall it months later. That's the whole loop.

  • 📄

    It's yours, as plain files

    Every paper is a PDF plus a small JSON file in a folder you pick. No proprietary database, no export ritual, no lock-in. Open it in Finder, back it up, grep it, sync it through iCloud — it's just files.

  • ⚡

    Fast and native

    A SwiftUI Mac app that launches instantly and stays out of the way. It opens PDFs in Preview instead of shipping a worse reader, so it stays small and quick.

  • 🔒

    Respects your attention

    No login, no cloud account, no upsell, no telemetry. One-time download, then it's a tool — not a service that watches you.

  • ✨

    Optional-smart

    If you have Claude Code or Ollama installed, Sift can auto-tag and summarize locally — but it works completely without them. The intelligence is a convenience, never a dependency.

Install in three steps

This build is signed ad-hoc (not notarized), so Gatekeeper warns on first launch. The right-click trick is needed once.

  1. 1

    Download the DMG

    Grab Sift-<version>.dmg from the latest release.

  2. 2

    Drag to Applications

    Open the DMG and drag Sift.app into your Applications folder.

  3. 3

    Right-click → Open

    Right-click the app, choose Open, then Open again to clear the first-launch warning.

What Sift doesn't do

Sift is a catalog, not a citation manager. If you need BibTeX export and DOI lookups, keep your citation tool. Being honest up front so the right people use it.

  • —

    No embedded PDF reader

    Deliberately. It opens PDFs in Preview, where macOS already does annotations well.

  • —

    No citation export

    No BibTeX, RIS, or DOI lookup. Possible future optional add-on.

  • —

    No cloud account or push sync

    It uses iCloud Drive for sync, so expect a few seconds of delay — never a login.
